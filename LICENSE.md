## Sobre este Projeto

Este projeto (FXdemo) é uma aplicação JavaFX desenvolvida como demonstração de conceitos avançados de desenvolvimento Java, incluindo:

- Arquitetura MVC com JavaFX
- Injeção de dependências manual
- Múltiplas formas de persistência (SQLite e CSV)
- Componentes JavaFX avançados
- Tratamento de eventos e validações
